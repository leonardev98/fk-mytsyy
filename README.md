# Mytsyy

Plataforma SaaS AI-first: de cero a tu primera venta en 30 días.

## Desarrollo

```bash
pnpm install
pnpm dev
```

Abre [http://localhost:3000](http://localhost:3000).

## Stack

- Next.js (App Router), TypeScript, Tailwind CSS
- Arquitectura hexagonal por módulos

Documentación del producto y arquitectura en `/docs`.
